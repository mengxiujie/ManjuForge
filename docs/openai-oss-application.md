## Project Introduction & Open Source Value

This MIT-0 licensed open-source project is a professional-grade AI Agent skill for short-video manju (漫剧) industrial production. It uniquely combines **script generation** and **professional storyboard creation** into a single unified toolchain.

**Core Innovation**: Unlike existing tools that only generate plain text scripts or basic storyboards, this project delivers a complete production-ready pipeline:

1. **Intelligent Script Generation**: Automatically converts one-sentence requirements, novel fragments, or long-form text into structured commercial scripts with character profiles, scene divisions, emotional arcs, and suspense optimization.

2. **Dual-Mode Storyboard Output**:
   - **Horizontal 16:9 Standard**: Traditional anime storyboard for human production crews
   - **Vertical 9:16 AI Execution**: Industry-first vertical short-drama storyboard with AI prompts, P0/P1/P2 priority system, 4-layer sound design, and fragmented timeline optimized for short-video platforms

3. **Professional Knowledge Base**: 13 world-class film analysis case studies (Inception, Matrix, Spirited Away, Your Name, Demon Slayer, Attack on Titan, etc.), vertical composition rules, AI challenge lens annotations, and complete end-to-end examples.

4. **Cross-Platform Compatibility**: Native support for Hermes Agent, OpenClaw, SiliconFlow, Cursor, with standardized JSON configuration for universal deployment.

The project fills a critical gap in the open-source community: there is no existing tool that combines industrial-grade script generation with professional storyboard creation, especially for the booming vertical short-video market. It directly serves individual creators, self-media studios, and small content teams seeking lightweight commercial production capabilities.
