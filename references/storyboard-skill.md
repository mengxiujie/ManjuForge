# 动漫分镜师 Skill

## Skill 概述

本 skill 为动漫制作提供专业的分镜脚本创作能力，支持两种模式：

1. **横屏标准版**（16:9）：传统动漫/长片分镜，面向人类摄制组
2. **竖屏AI执行版**（9:16）：短剧/竖屏分镜，面向AI生成器，含提示词+优先级+声音分层

当用户需要将动漫剧本转化为可视化分镜时，根据项目类型自动选择模式。

## 使用场景

以下情况应使用本 skill：
- 用户提供动漫剧本或故事梗概，要求生成分镜脚本
- 用户描述场景画面，需要拆解为分镜
- 用户需要动漫分镜模板或格式指导
- 用户询问动漫分镜的相关知识
- **竖屏短剧/AI生成分镜**（新增触发词：竖屏/短剧/AI分镜/执行版/碎片/提示词）

## 模式判断

| 条件 | 模式 |
|------|------|
| 明确说"竖屏"/"短剧"/"9:16"/"AI执行版"/"碎片化" | **竖屏AI执行版** |
| 项目时长≤5分钟/集 | **竖屏AI执行版** |
| 面向AI视频/图像生成 | **竖屏AI执行版** |
| 其他情况 | **横屏标准版** |

## 一、横屏标准版（16:9）

### 分镜表必含字段

| 字段 | 说明 | 示例 |
|------|------|------|
| 镜号 | 镜头序号 | 01、02、03 |
| 场景 | 场景编号/名称 | S01-室内教室 |
| 景别 | 镜头距离 | 远景/全景/中景/近景/特写/大特写 |
| 运镜 | 镜头运动方式 | 推、拉、摇、移、跟、升降、固定 |
| 时长 | 镜头持续时间 | 5秒、8秒 |
| 画面描述 | 角色动作、神态、环境 | 夕子走进教室，阳光从窗外照入 |
| 台词/独白 | 角色对话或内心独白 | "今天也要加油！" |
| 音效/音乐 | 背景音乐或音效提示 | BGM-欢快、门铃声 |
| 情绪氛围 | 画面整体调性 | 温馨、紧张、压抑、欢快 |
| 色调建议 | 整体色彩倾向 | 暖色调、冷色调、复古色 |

### 动漫特有的分镜要点

1. **角色定位（卡位）**：标注角色在画面中的位置和朝向
2. **关键帧设计**：标注动作的关键瞬间（原画点）
3. **表情特写**：标注重要角色表情变化
4. **特效提示**：标注需要添加的特效（光效、粒子等）
5. **口型同步**：标注重要对话的口型要点
6. **速度线/效果线**：标注动态效果增强

### 输出格式（横屏）

#### Markdown 表格格式

```markdown
| 镜号 | 场景 | 景别 | 运镜 | 时长 | 画面描述 | 台词/音效 | 情绪 | 色调 |
|------|------|------|------|------|----------|-----------|------|------|
| 01 | S01-教室 | 中景 | 固定 | 5s | 夕子走进教室，阳光透过窗户 | "早上好！" | 温馨 | 暖色 |
```

#### 详细描述格式

```
【第01镜】
- 场景：S01-室内教室 日
- 景别：中景
- 运镜：固定
- 时长：5秒
- 画面：夕子背着书包走进教室，阳光从窗外照入，在地面形成光斑。教室里已有几位同学在做自己的事。
- 台词：夕子（元气满满）："大家早上好！"
- 音效：开门声、BGM-轻快温馨
- 情绪：温馨、活力
- 色调：暖黄色调，阳光明媚
- 备注：夕子进门时使用夸张的开门动作，增加喜剧效果
```

### 分镜表必含字段（AI执行版）

| 字段 | 代号 | 说明 | 示例 |

| 碎片编号 | × | 段落+序号 | A3 / B1 / H2 |
| 段落 | 段 | 段落标识 | A/B/C/D/E/F/G/H/I |
| 时间轴 | 时间 | 入点→出点 | 00:18→00:25 |
| 景别/运镜 | 景 | 景别+运镜组合 | WS·垂直下降 / ECU·微推 |

| 声音设计 | 声 | 四层分层标注 🎵🔊💓🫧 | 🎵尺八·单音 🔊远海浪 💓无 🫧无 |

| 横屏景别 | 竖屏等效 | 人物占比 |

| 远景WS | 极远景EWS | 3-8% |

### 竖屏运镜（横屏无对应）

| 运镜 | 代号 | 情绪效果 |
|------|:----:|----------|
| 垂直下降 | V-DOWN | 宏大→压迫/发现 |
| 垂直上升 | V-UP | 孤独→释放/消逝 |
| 仰拍坠落 | LOOK-UP | 恐惧/无力 |
| 俯拍下陷 | LOOK-DOWN | 宿命/窒息 |
| 主观旋转 | POV-ROT | 极度恐惧/失控 |
| 垂直推进 | V-PUSH | 聚焦/侵入 |
| 垂直拉远 | V-PULL | 孤独/消逝/死亡 |

### 声音设计四层标注

| 层级 | 图标 | 功能 |
|------|:----:|------|
| 音乐层 | 🎵 | 情绪定调+主题动机 |
| 环境层 | 🔊 | 空间感知+氛围营造 |
| 生理层 | 💓 | 角色内在状态（心跳/呼吸） |
| 细节层 | 🫧 | 微观质感+触觉感 |

标注格式：
```
🎵 尺八·单音悠长（3秒绕梁）
🔊 极远海浪（白噪音级）
💓 心跳·加速
🫧 蝴蝶振翅
```

> 完整声音设计规范见 `references/knowledge_base/sound_design_layers.md`

| 优先级 | 含义 | 时机 |
|--------|------|------|
| P0 | 必须就位 | 缺少=无法开拍 |
| P1 | 应就位 | 缺少=可开工但需补 |
| P2 | 后续迭代 | 缺少=上线后补 |

| 等级 | 标注 | 说明 |

## 分镜创作流程

### 横屏标准版流程

1. **理解剧本**：通读剧本，把握整体风格和叙事节奏，识别关键场景和情绪转折点
2. **镜头规划**：根据叙事需要规划镜头数量，选择景别组合，设计镜头运动
3. **画面描述**：用简洁清晰的语言描述每个画面，标注角色位置、动作和表情
4. **完善细节**：添加台词、音效、音乐提示，标注特效需求，检查节奏和情绪连贯性

### 竖屏AI执行版流程

1. **理解剧本+确定碎片化策略**：
   - 将剧本拆分为2-4个情绪段落（30-90秒/段）
   - 每段拆分为10-15秒碎片
   - 确定钩子段（前3秒）和收束段（最后10秒）
   > 详见 `references/knowledge_base/vertical_short_drama.md`

2. **时间轴规划**：
   - 从00:00开始，逐碎片编排入点→出点
   - 碎片间无缝衔接，过渡碎片单独标注
   - 总时长校验（≤5分钟）

3. **画面+声音+AI提示词三位一体编写**：
   - 每碎片：画面描述（含色值/构图/情绪锚点）+ 四层声音 + AI提示词（含优先级）
   - AI挑战镜头标注（S/A/B级）
   - 跨集符号锚点标注
   > 详见 `references/knowledge_base/ai_execution_fields.md` 和 `references/knowledge_base/sound_design_layers.md`

4. **资产清单+自检**：
   - 编制资产清单（附录A），绑定碎片
   - AI执行版专项自检（10项新增检查）
   - 碎片时长/总时长/静默时长校验
   > 详见 `references/knowledge_base/ai_execution_fields.md` 第八节

## 常见动漫分镜风格

### 热血战斗类
- 强调动作流畅感和力度
- 多用特写和快速剪辑
- 注重速度线和效果线
- 色调对比强烈

### 日常校园类
- 节奏轻松舒缓
- 景别以中近景为主
- 注重角色表情细腻变化
- 色调明亮清新

### 科幻机战类
- 强调机械细节和科技感
- 多用远景展现巨大感
- 注重光效和粒子效果
- 色调偏冷，蓝色/银色为主

### 奇幻冒险类
- 注重世界观呈现
- 场景变化丰富
- 魔法/特效表现重要
- 色调根据魔法属性变化

### 竖屏神话短剧类（新增）
- 前3秒必须有视觉钩子
- 画面驱动为主，极简/无台词
- 碎片化时间轴（10-15秒/碎片）
- 声音四层分层设计（🎵🔊💓🫧）
- 静默作为情绪武器（3-5秒）
- 符号锚点跨集管理
- 悬念截断式结尾
