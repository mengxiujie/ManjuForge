## 一、整体分镜风格特点

### 1.1 巨人美学
- **巨大感**：巨人vs人类的体型差

- **黑暗**：压抑的世界观

- **刀刃**：切割的锐利感
- **血液**：红色的冲击

## 三、分镜技巧总结

### 3.1 巨人压迫感公式
```
远景 → 中景 → 特写
(建立) (接近) (压迫)
```

### 3.2 立体机动战斗节奏
```
瞄准(1s) → 冲刺(1s) → 切割(0.5s) → 脱离(1s)
```

### 3.3 战斗类型分镜
| 战斗 | 特点 | 镜头 |
|------|------|------|
| 巨人vs人类 | 体型差 | 仰视/俯视 |
| 巨人vs巨人 | 巨物互殴 | 远景+震动 |
| 立体机动 | 高速 | 速度线+近景 |
| 智斗 | 悬念 | 特写+推理 |

### 3.4 情感分镜
- 牺牲时的慢动作
- 胜利的短暂快乐
- 真相的震撼
- 角色的成长弧线
