# 拉片案例06：《你的名字》君の名は (2016)

**导演**：新海诚
**类型**：爱情/奇幻/青春
**核心分镜特点**：极致风景美学、时空交错叙事、彗星坠落、细腻情感

---

## 一、整体分镜风格特点

### 1.1 新海诚视觉美学
- **真实背景**：使用实际取景的照片作为背景
- **光影效果**：逆光、散光、唯美夕阳
- **云**：动态云层是标志元素
- **线条**：清晰锐利的高对比

### 1.2 双线空间
| 空间 | 地点 | 色调 | 特点 |
|------|------|------|------|
| 东京 | 都市 | 蓝/灰/霓虹 | 便利、忙碌、现代 |
| 岐阜县 | 乡村 | 绿/蓝/自然 | 传统、宁静、古老 |
| 梦（交换） | 两地 | 交融 | 模糊、眩晕 |

## 三、分镜技巧总结

### 3.1 时空交错技巧
```
现实线 → 梦/回忆 → 现实改变
       ↓
    另一条时间线
```

### 3.2 风景与情感
```
风景不只是背景 → 情感的外化
云 = 心情的流动
雨 = 悲伤/分别
夕阳 = 魔幻/永恒
```

### 3.3 情绪节奏公式
```
日常 → 意外 → 适应 → 危机 → 拯救 → 失去 → 重逢
```

### 3.4 对称构图
- 楼梯相遇的对称
- 天空与大地的对应
- 两人空间的呼应

## 四、对动漫分镜的借鉴价值

### 4.1 可应用于
- **时空穿越题材**
- **爱情动画**
- **青春成长**
- **风景展示**

### 4.2 学习要点
1. 真实背景+动画前景
2. 风景渲染情绪
3. 时空交叉剪辑
4. 经典台词设计

### 4.3 核心技巧
- 彗星：美丽与危险并存
- 楼梯：命运的相遇空间
- 黄昏：魔幻时刻
- 留下"名字"的悬念
- 开放式但温暖的结局
