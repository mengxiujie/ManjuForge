## 三、分镜技巧总结

### 3.1 水墨战斗分镜
```
起手（蓄力）→ 释放 → 效果 → 影响 → 结束
  ↓        ↓      ↓      ↓
 水墨晕染  色彩  扩散  延续
```

### 3.2 五行特效视觉化
| 五行 | 形态 | 颜色 | 运动 |
|------|------|------|------|
| 火 | 燃烧/爆炸 | 红/金 | 向上/扩散 |
| 水 | 流动/冰冻 | 蓝/青 | 流淌/凝固 |
| 雷 | 闪电/速度 | 紫/白 | 直线/跳跃 |
| 风 | 旋转/切割 | 绿/白 | 螺旋/扩散 |
| 土 | 防御/厚重 | 棕/黄 | 凝固/防御 |

### 3.3 情绪节奏
```
压抑 → 展示 → 冲突 → 爆发 → 反思
  |      |      |      |      |
  远景  特写   中景  远景   远景
```

## 四、对动漫分镜的借鉴价值

### 4.1 可应用于
- **中国奇幻**
- **武侠战斗**
- **五行/道法题材**
- **传统文化题材**

### 4.2 学习要点
1. 水墨风格的动作表现
2. 五行法术的视觉化
3. 传统文化元素融入
4. 战斗的节奏感
5. 贪婪/欲望的主题表达

### 4.3 核心技巧
- 水墨不只是风格，是意境
- 战斗要分阶段
- 特效要有文化根基
- 群像要有序
- 主题要深刻
