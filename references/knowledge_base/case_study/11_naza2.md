# 拉片案例11：《哪吒之魔童闹海》 (2025)

**导演**：饺子
**类型**：神话/动作/奇幻/动画电影
**核心分镜特点**：神话想象力、极致特效、水系法术视觉、中国传统美学融合现代风格

---

## 一、整体分镜风格特点

### 1.1 视觉美学
- **东方奇幻**：中国传统神话美学

## 三、分镜技巧总结

### 3.1 战斗分镜公式
```
危机（远景）→ 对峙（中景）→ 爆发（特写+特效）→ 对抗（远景）→ 转折（特写）→ 结局（远景）
```

### 3.2 特效与角色结合
```
角色动作 + 特效包裹 + 环境变化 + 能量展示
```

### 3.3 情绪节奏
```
平缓 → 危机 → 绝望 → 希望 → 爆发 → 解决
```

### 3.4 群像分镜技巧
- 用远景展示整体
- 用中景展示群体
- 用特写展示个体
- 交叉剪辑

## 四、对动漫分镜的借鉴价值

### 4.1 可应用于
- **神话改编**
- **中国奇幻**
- **热血战斗**
- **英雄成长**

### 4.2 学习要点
1. 传统文化元素的现代化表达
2. 极致特效与角色情感的结合
3. 反派形象的立体化
4. 伙伴羁绊的情感核心
5. 群像与个体的平衡

### 4.3 核心技巧
- 战斗要有"势"的概念
- 特效要服务情感
- 反派不是工具人
- 成长需要代价
- 中国神话的视觉化
