# 拉片案例10：《中国奇谭》 (2023)

**导演**：陈廖宇（总导演）、各集导演
**类型**：动画短片集/中国风/奇幻
**核心分镜特点**：中国传统美学、多样化风格、东方叙事哲学、文化符号创新

---

## 一、整体分镜风格特点

### 1.1 各集风格概览

| 篇章 | 风格 | 色调 | 艺术形式 |
|------|------|------|----------|
| 小妖怪的夏天 | 传统水墨 | 墨色/暖橙 | 平面水墨 |
| 鹅鹅鹅 | 素描/水墨 | 黑白/红 | 中国画留白 |
| 林林 | 油画/3D | 冷灰/棕 | 写实油画 |
| 乡村巴士带走了四千人 | 拼贴/现代 | 彩色 | 拼贴艺术 |
| 小满 | 剪纸/定格 | 红色/暖色 | 剪纸动画 |
| 飞鸟与鱼 | 水彩/梦幻 | 蓝紫/粉 | 水彩渲染 |

### 1.2 核心美学特征
- **留白**：大量留白创造意境
- **意境**：情景交融的诗意表达
- **符号**：中国传统视觉元素
- **叙事**：东方哲学的含蓄表达

## 三、分镜技巧总结

### 3.1 中国风分镜特点
```
留白 → 意境
疏密 → 节奏
虚实 → 层次
动静 → 情绪
```

### 3.2 情绪表达方式
- **含蓄**：不直接表现情感，通过意境
- **留白**：给观众想象空间
- **象征**：用物象代替直说
- **意境**：情景交融

### 3.3 视觉符号系统
| 符号 | 含义 |
|------|------|
| 山/水 | 意境/自然 |
| 云/雾 | 神秘/变化 |
| 红 | 热情/危险 |
| 留白 | 想象/空白 |

## 四、对动漫分镜的借鉴价值

### 4.1 可应用于
- **中国奇幻/神话改编**
- **艺术短片**
- **水墨/传统艺术风格**
- **含蓄情感表达**

### 4.2 学习要点
1. 传统美学融入现代动画
2. 留白创造意境
3. 符号化表达
4. 多样化艺术风格探索
5. 东方叙事的含蓄美

### 4.3 核心技巧
- 不是"画"出来的，是"写"出来的
- 情绪不直说，通过意境传达
- 传统元素现代化表达
- 每个短片都是独立的艺术探索
