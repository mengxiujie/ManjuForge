# 拉片案例03：《千与千寻》千と千尋の神隠し (2001)

**导演**：宫崎骏
**类型**：奇幻/冒险/成长
**核心分镜特点**：细腻的日常细节、温暖的色调变化、流畅的自然运动、充满想象力的奇幻世界

---

## 一、整体分镜风格特点

### 1.1 宫崎骏分镜美学
- **手绘质感**：温暖的笔触，柔和的线条
- **自然主义**：真实的物理法则，动植物细节
- **情感细腻**：通过小动作传达内心
- **空间纵深**：多层背景创造深度感

### 1.2 色调系统
| 场景 | 色调 | 情绪 |
|------|------|------|
| 隧道/进入异世界 | 蓝灰→暖黄 | 神秘→温暖 |
| 汤屋 | 红金/暖色 | 奢华、热闹 |
| 锅炉房 | 暖橙/棕 | 温暖、神秘 |
| 河川（白龙） | 蓝绿/透明 | 纯净、神秘 |
| 电车场景 | 淡黄/黄昏 | 孤独、温柔 |

## 三、分镜技巧总结

### 3.1 日常细节的魔法
```
普通动作 → 添加奇幻元素 → 变成魔法
例：开门 → 巨大生物开门
    吃饭 → 变成猪
    打扫 → 变成劳动
```

### 3.2 情绪节奏公式
```
日常 → 意外 → 危机 → 成长 → 解决 → 回归
```

### 3.3 空间构建技巧
- **入口**：门、隧道、楼梯等过渡空间
- **内部**：拥挤、复杂、多层
- **高处**：一览众山小的视角
- **自然**：与外界的连接

### 3.4 角色运动分镜
- **走路**：自然的起伏
- **跑步**：根据情绪调整速度
- **飞**：遵循物理法则（滑翔感）
- **停顿**：重要的情感时刻

---

## 四、对动漫分镜的借鉴价值

### 4.1 可应用于
- **奇幻冒险类**
- **日常治愈类**
- **成长题材**
- **异世界冒险**

### 4.2 学习要点
1. 日常细节让奇幻更可信
2. 色调随情绪和场景变化
3. 给角色留"停顿"时间
4. 用空间变化展示心理变化

### 4.3 核心技巧
- **渐变**：色调、情绪的平滑过渡
- **对比**：大小、明暗、善恶对比
- **循环**：自然元素的循环（风、水、火）
- **留白**：给观众想象空间
