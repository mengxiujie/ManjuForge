# 拉片案例09：《万神殿》Pantheon (2022-)

**导演**：Craig McKacken
**类型**：科幻/赛博朋克/悬疑
**核心分镜特点**：现实与数字世界的视觉分屏、意识上传的视觉表现、冷静克制的科幻美学

---

## 一、整体分镜风格特点

### 1.1 双重世界视觉系统

| 世界 | 色调 | 画面特点 | 镜头风格 |
|------|------|----------|----------|
| **现实世界** | 冷蓝/灰/自然光 | 真实家庭场景、写实风格 | 稳定、手持感 |
| **UI/数字世界** | 亮白/绿/蓝 | 数据流、界面、网格 | 锐利、扁平、快速 |

### 1.2 核心视觉元素
- **分屏表现**：现实与数字世界同时展示
- **数据可视化**：意识上传过程的视觉化
- **UI叠加**：数字信息覆盖现实画面
- **扫描线效果**：数字扫描过程

## 三、分镜技巧总结

### 3.1 现实/数字世界切换

| 转换方式 | 效果 |
|----------|------|
| 扫描线过渡 | 数字扫描感 |
| 渐变叠加 | 意识进入 |
| 画面分割 | 同时展示两世界 |
| UI覆盖 | 数字信息叠加 |

### 3.2 科幻信息展示
```
数据出现 → 放大特写 → 角色反应 → 理解 → 行动
```

### 3.3 情绪节奏
- **悬疑**：慢节奏 + 特写 + 冷静色调
- **动作**：快速切换 + 跟随镜头
- **情感**：长镜头 + 面部特写
