# 分镜理论手册

本手册涵盖动漫分镜的核心理论，包括叙事技巧、情绪表达、节奏控制等，是分镜创作的指导思想。

---

## 第一章：分镜基础理论

### 1.1 分镜的定义

**分镜（Storyboard）** 是将剧本转化为可视画面的中间创作环节，它决定了：
- 镜头如何叙述故事
- 画面如何传达情绪
- 节奏如何影响观众

### 1.2 分镜的核心功能

| 功能 | 说明 |
|------|------|
| 可视化 | 将文字转化为画面 |
| 叙事 | 决定"看什么"和"怎么看" |
| 节奏 | 控制信息传递的速度 |
| 情绪 | 引导观众的心理感受 |
| 沟通 | 让团队理解导演意图 |

### 1.3 分镜与其他环节

```
剧本 → 分镜 →  Layout → 原画 → 动画 → 后期
  ↓       ↓       ↓       ↓      ↓       ↓
文字   画面    空间    关键帧  中间帧   完成
```

## 第二章：叙事理论

### 2.1 镜头即句子

类比语法：
- **单词** = 景别（特写、近景）
- **短语** = 组合镜头
- **句子** = 单个镜头
- **段落** = 场景

### 2.2 叙事视角

#### 第一人称（POV）
- **特点**：观众代入角色
- **优势**：沉浸感、紧张感
- **劣势**：信息有限
- **适用**：恐怖、悬疑、内心戏

#### 第三人称
- **特点**：客观展示
- **优势**：信息全、客观
- **劣势**：距离感
- **适用**：大多数类型

#### 上帝视角
- **特点**：全知全能
- **优势**：信息量大
- **劣势**：缺乏悬念
- **适用**：史诗、战争

### 2.3 视线引导

#### 视觉重心
```
画面重心 = 观众最先看的地方
```

**引导方法**：
1. 位置：画面上方/左侧
2. 运动：动的物体吸引注意
3. 对比：亮/大/色对比
4. 焦距：清晰 vs 模糊
5. 面部：眼睛是重心

#### 90度/180度规则

**90度规则**：
- 摄像机保持在角色同一侧
- 避免角色在画面中左右颠倒
- 保持空间连贯性

**180度规则**：
- 摄像机保持在角色"轴线"一侧
- 保持"屏幕方向"一致
- 违反会制造混淆

## 第三章：情绪理论

### 3.1 情绪层次

#### 表面情绪 vs 深层情绪
```
表面：角色表现出的（微笑）
深层：角色真实的（悲伤）
```

**分镜任务**：
- 表面通过表情/动作展示
- 深层通过环境/色调暗示

### 3.2 情绪曲线

#### 基本情绪曲线
```
高潮
  |     /\
  |    /  \
  |   /    \________
  |  /
  |/
起始点
```

#### 动漫常见情绪曲线

| 曲线类型 | 描述 | 适用 |
|----------|------|------|
| 渐升型 | 逐步紧张到高潮 | 战斗、悬疑 |
| 波浪型 | 起伏多次 | 冒险、成长 |
| V型 | 低谷后反转 | 悲剧转喜剧 |
| 倒V型 | 高潮后跌落 | 喜剧转悲剧 |
| 平台型 | 持续紧张 | 马拉松式战斗 |

### 3.3 情绪工具箱

| 工具 | 作用 |
|------|------|
| 景别 | 特写=情绪强，远景=情绪弱 |
| 色调 | 暖=积极，冷=消极 |
| 音乐 | 配合画面加强情绪 |
| 节奏 | 快=紧张，慢=放松 |
| 静止 | 停顿增加张力 |

### 3.4 情绪分镜公式

#### 悲伤场景
```
1. 环境建立（阴雨天）
2. 角色近景（表情低落）
3. 触发物特写（遗物/照片）
4. 回忆闪回（暖色调）
5. 回到现实（特写泪眼）
6. 远景收尾（孤独背影）
```

#### 战斗爆发
```
1. 危机时刻（特写紧张）
2. 内心独白/回忆（坚定眼神）
3. 变身/蓄力（中景→远景）
4. 技能特写（特效满满）
5. 攻击瞬间（远景+冲击）
6. 敌人反应（震惊特写）
7. 胜利/倒下（远景）
```

## 第四章：节奏理论

### 4.1 节奏的定义

**节奏** = 信息传递的速度 + 画面变化的频率

| 节奏 | 镜头数/分钟 | 适用场景 |
|------|--------------|----------|
| 极慢 | <30 | 日常、文艺 |
| 慢 | 30-60 | 情感、对话 |
| 中 | 60-90 | 剧情推进 |
| 快 | 90-150 | 动作 |
| 极快 | >150 | 战斗高潮 |

### 4.2 节奏变化

#### 黄金法则：变化
```
不能一直快 → 需要停顿
不能一直慢 → 需要推进
```

#### 节奏公式
```
平静(慢) → 紧张(快) → 高潮(极快) → 结束(收束)
  ↓
对比制造冲击
```

### 4.3 节奏控制工具

| 工具 | 减速 | 加速 |
|------|------|------|
| 镜头时长 | 长 | 短 |
| 景别 | 远景 | 特写 |
| 剪辑 | 淡入淡出 | 直接切换 |
| 动作 | 完整展示 | 省略 |
| 音乐 | 完整 | 切掉前奏 |

### 4.4 节奏失误

| 问题 | 原因 | 解决 |
|------|------|------|
| 拖沓 | 镜头太长 | 减少时长 |
| 仓促 | 镜头太短 | 增加时长 |
| 无聊 | 节奏单一 | 增加变化 |
| 混乱 | 节奏太快 | 适当停顿 |

## 第五章：空间理论

### 5.1 空间建立

#### 空间信息层次
```
1. 建筑/地点（远景）
2. 区域/房间（全景）
3. 人物位置（中景）
4. 细节（特写）
```

#### 首次空间建立规则
- 使用远景或全景
- 保持镜头足够长（3秒+）
- 展示关键元素
- 让观众"了解"这个空间

### 5.2 空间转换

#### 无缝转换
- 匹配动作（A开车出门切到A下车）
- 匹配物体（抛出的球切到接住）
- 匹配视线（看窗外切到窗外）

#### 有缝转换
- 黑场：硬分隔
- 叠化：时间流逝
- 转场特效：风格化

### 5.3 空间情绪

| 空间 | 情绪 |
|------|------|
| 封闭 | 压抑、紧张 |
| 开放 | 自由、孤独 |
| 熟悉 | 安全、怀旧 |
| 陌生 | 紧张、好奇 |
| 高处 | 权力、晕眩 |
| 低处 | 渺小、无力 |

## 第六章：角色理论

### 6.1 角色展示

#### 首次登场分镜
```
1. 远景：角色出现在环境中
2. 中景：展示全身/姿态
3. 近景：突出特征/服装
4. 特写：面部特征/表情
```

#### 角色识别元素
- 外貌特征（发型、眼睛、配饰）
- 配色（服装、发色）
- 姿态/站姿
- 习惯动作

### 6.2 角色调度

#### 画面位置意义
```
画面左侧 → 过去/消极
画面右侧 → 未来/积极
画面上方 → 权力/积极
画面下方 → 无力/消极
```

#### 角色关系表现
- **对立**：中间有障碍物
- **亲密**：近距离，无障碍
- **主从**：位置高低
- **三角**：三人在画面中

### 6.3 角色动作

#### 动作分级
| 级别 | 动作 | 分镜密度 |
|------|------|----------|
| 微动作 | 表情、眼神 | 重点特写 |
| 小动作 | 手势、姿态 | 中景展示 |
| 大动作 | 走路、跑步 | 全身 |
| 关键动作 | 战斗、变身 | 全方位 |

## 第七章：类型理论

### 7.1 热血战斗类

#### 分镜特点
- 高速剪辑（1-3秒/镜头）
- 多角度展示
- 特效密集
- 情绪爆发

#### 公式
```
危机 → 回忆 → 觉悟 → 爆发 → 胜利
  |      |       |      |
  特写  慢动作  远景  特写+
```

### 7.2 日常治愈类

#### 分镜特点
- 长镜头为主
- 景别变化小
- 色调温暖
- 细节丰富

#### 公式
```
日常 → 小问题 → 解决 → 温馨
  |        |       |
  远景    特写    远景
```

### 7.3 科幻奇幻类

#### 分镜特点
- 建立世界观
- 解释性展示
- 大场面展示
- 特殊视觉风格

#### 公式
```
世界 → 设定 → 冲突 → 解决
  |      |       |
  远景  动画    全景
```

### 7.4 悬疑推理类

#### 分镜特点
- 信息控制（选择性展示）
- 引导思考
- 关键证据特写
- 紧张节奏

#### 公式
```
线索 → 推理 → 误导 → 真相
  |      |       |
  特写  中景    特写
```

## 第八章：实战技巧

### 8.1 分镜顺序

#### 标准顺序
```
1. 读剧本，理解故事
2. 划分场景
3. 确定每场情绪
4. 分配镜头数量
5. 绘制关键帧
6. 填充过渡镜头
7. 标注台词/音效
8. 检查节奏
```

### 8.2 快速分镜技巧

#### 1分钟场景模板
```
[0-5s] 远景建立
[5-15s] 中景推进
[15-30s] 核心部分（动作/对话）
[30-45s] 关键转折
[45-60s] 收束
```

#### 对话分镜模板
```
A说话：A中景+B反应
B说话：B中景+A反应
(交替进行...)
关键台词：双人中景
结束：特写反应
```

### 8.3 常见问题解决

| 问题 | 解决 |
|------|------|
| 不知道从哪开始 | 从最关键的3个镜头开始 |
| 节奏太平 | 加入特写和远景交替 |
| 情感不够 | 加入角色反应镜头 |
| 动作不清楚 | 拆分动作，添加关键帧 |
| 转换太硬 | 加入过渡镜头 |

## 附录：理论速查

### 叙事原则
- 信息：只展示需要的
- 清晰：让观众知道看哪里
- 变化：避免重复
- 节奏：有快有慢

### 情绪原则
- 对应：情绪匹配画面
- 积累：逐步加强
- 释放：高潮后要收束
- 留白：不是所有都要解释

### 技巧原则
- 90度/180度规则
- 视线引导
- 景别变化
- 节奏变化
- 空间连贯
