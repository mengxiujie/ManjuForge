# 分镜质量自检清单

本清单用于分镜完成后的质量检查，确保分镜的专业性和完整性。

---

## 第一部分：完整性检查

### 1.1 基础信息检查

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| [ ] 每场戏有场景编号 | 格式统一：S01、S02... | 必填 |
| [ ] 每场戏有时间标注 | 日/夜/晨/昏 | 必填 |
| [ ] 每场戏有地点标注 | 室内/室外+具体地点 | 必填 |
| [ ] 每场戏有情绪标注 | 紧张/温馨/悲伤... | 必填 |
| [ ] 每场戏有色调标注 | 暖色/冷色... | 建议 |

### 1.2 镜头信息检查

| 检查项 | 要求 | 优先级 |
|--------|------|--------|
| [ ] 每个镜头有镜号 | 连续编号 | 必填 |
| [ ] 每个镜头有景别 | 远/全/中/近/特/大特 | 必填 |
| [ ] 每个镜头有运镜 | 固定/推/拉/摇... | 必填 |
| [ ] 每个镜头有时长 | 单位：秒 | 必填 |
| [ ] 每个镜头有画面描述 | 清晰可执行 | 必填 |
| [ ] 对话有台词 | 标注说话者 | 建议 |
| [ ] 有音效/音乐提示 | BGM/音效 | 建议 |

### 1.3 数量检查

| 检查项 | 标准 |
|--------|------|
| [ ] 总镜头数合理 | 约2-4秒/镜头 |
| [ ] 景别有变化 | 至少有2种以上景别 |
| [ ] 运镜有变化 | 至少有2种以上运镜 |
| [ ] 时长有变化 | 避免所有镜头时长相同 |

## 第八部分：自我评审

### 评分系统（每项1-5分）

| 评估项 | 分数(1-5) | 说明 |
|--------|------------|------|
| 完整性 |  | 1=缺失严重，5=完美 |
| 技术性 |  | 1=错误频出，5=专业 |
| 艺术性 |  | 1=平淡无奇，5=精彩 |
| 可执行性 |  | 1=无法执行，5=清晰明确 |
| 创新性 |  | 1=平庸老套，5=有新意 |

**总分：/25**

- 20-25：优秀，可直接进入下一环节
- 15-19：良好，稍作修改即可
- 10-14：一般，需要较大修改
- <10：不合格，需要重做

## 第九部分：常见问题修正

### 问题与解决方案

| 常见问题 | 解决方案 |
|----------|----------|
| 镜头太平淡 | 添加景别变化，添加特写 |
| 节奏太单调 | 加入快慢交替，添加停顿 |
| 情感不够 | 加入角色反应镜头，加入环境配合 |
| 动作不清楚 | 拆分关键动作，加入分解镜头 |
| 转换太硬 | 加入过渡镜头，使用叠化 |
| 信息不够 | 加入说明镜头，加入特写 |
| 信息太多 | 简化非关键内容，减少镜头 |

## 第十部分：检查流程

### 快速检查流程（30分钟）

```
Step 1: 基础检查 (5分钟)
  - 场景信息完整
  - 镜头信息完整
  
Step 2: 技术检查 (10分钟)
  - 叙事通顺
  - 镜头语言正确
  - 技术问题

Step 3: 艺术检查 (10分钟)
  - 情绪
  - 节奏
  - 画面

Step 4: 细节检查 (5分钟)
  - 台词
  - 音效
  - 特效
```

### 详细检查流程（2小时）

```
Day 1: 自我检查
  - 按清单逐项检查
  - 记录问题
  
Day 2: 修改完善
  - 解决问题
  - 调整节奏

Day 3: 最终检查
  - 快速浏览
  - 细节打磨
```
