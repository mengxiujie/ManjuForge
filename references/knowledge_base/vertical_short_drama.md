# 竖屏短剧分镜专项

> 本文档为竖屏9:16短剧（3-5分钟/集）的分镜专项规则，补充横屏16:9通用规则在竖屏场景下的差异与适配。

---

## 一、竖屏9:16构图规则

### 1.1 画幅特性

| 维度 | 横屏16:9 | 竖屏9:16 | 影响 |
|------|:--------:|:--------:|------|
| 宽高比 | 1.78:1 | 0.56:1 | 竖屏水平空间压缩56% |
| 视觉重心 | 中心偏右/左 | **垂直中轴** | 竖屏构图以纵轴为骨架 |
| 自然视线 | 水平扫描 | **自上而下** | 竖屏观众先看上方 |
| 环境展示 | 宽→水平延展 | **深→垂直纵深** | 竖屏善用天空/地面层次 |
| 人物占比 | 同景别占比小 | 同景别**占比大** | 竖屏中景≈横屏近景 |

### 1.2 竖屏三分法

```
┌─────────────┐
│   天空/高位  │ ← 上1/3：环境/氛围层
│             │
├─────────────┤
│             │ ← 中1/3：叙事/人物层
│   视觉重心  │    （核心动作/表情）
│             │
├─────────────┤
│   地面/低位  │ ← 下1/3：前景/纵深层
│             │
└─────────────┘
```

### 1.3 竖屏景别换算表

| 横屏景别 | 竖屏等效景别 | 人物占比 | 用途 |
|----------|:----------:|:-------:|------|
| 远景WS | **极远景EWS** | 3-8% | 环境/悬崖/海面/天空 |
| 全景FS | **远景WS** | 15-25% | 全身+环境 |
| 中景MS | **中近景MCU** | 35-50% | 胸部以上，竖屏叙事主力 |
| 近景CU | **特写CU** | 50-70% | 面部/手持物 |
| 特写CU | **大特写ECU** | 70-90% | 瞳孔/手指/发簪 |
| 大特写ECU | **微距MACRO** | 90-100% | 瞳孔纹理/红线/碎光 |

### 1.4 竖屏构图禁区

| 禁区 | 原因 | 替代方案 |
|------|------|----------|
| 水平双人正反打 | 画面左右空间不足 | **上下分层构图**：一人在上1/3，一人在下1/3 |
| 宽幅横移展示 | 水平空间不够 | **垂直升降**代替水平横移 |
| 多人群像 | 人物过小看不清 | **纵深分层**：前景1人+中景1-2人+远景剪影 |
| 左右对称构图 | 竖屏纵向感被削弱 | **上下对称/镜像**代替左右对称 |

## 二、竖屏运镜字典

### 2.1 竖屏核心运镜（横屏无对应）

| 运镜名称 | 代号 | 描述 | 情绪效果 | 适用场景 |
|----------|:----:|------|----------|----------|
| **垂直下降** | V-DOWN | 镜头从天空向地面垂直下降 | 宏大→压迫/发现 | 悬崖/高楼/深渊 |
| **垂直上升** | V-UP | 镜头从地面向天空垂直上升 | 孤独→释放/消逝 | 死亡/远去/祈祷 |
| **仰拍坠落** | LOOK-UP | 从下方仰视人物坠落 | 恐惧/无力 | 坠崖/掉落/下沉 |
| **俯拍下陷** | LOOK-DOWN | 从上方俯视人物被吞没 | 宿命/窒息 | 水下/深陷/被包围 |
| **主观旋转** | POV-ROT | 第一人称旋转（天/地交替） | 极度恐惧/失控 | 坠落/眩晕/惊恐 |
| **垂直推进** | V-PUSH | 沿纵轴从远到近推进 | 聚焦/侵入 | 从远景到面部特写 |
| **垂直拉远** | V-PULL | 沿纵轴从近到远拉远 | 孤独/消逝/死亡 | 人物变小的终极远景 |

### 2.2 竖屏与横屏运镜对照

| 横屏运镜 | 竖屏替代 | 说明 |
|----------|----------|------|
| 水平摇镜(PAN) | **垂直摇镜(TILT)** | 上下摇代替左右摇 |
| 横移跟拍(FOLLOW-H) | **纵深跟拍(FOLLOW-D)** | 前后跟随代替左右跟随 |
| 环绕拍摄(ORBIT) | **V-DOWN + 旋转** | 竖屏中环绕用下降+慢旋组合 |
| 横移揭示(REVEAL-H) | **垂直揭示(REVEAL-V)** | 下降揭示（天空→人物→悬崖） |
| 宽幅平移(PANORAMA) | **深焦分层(DEEP-FOCUS)** | 用景深分层代替宽幅展示 |

### 2.3 竖屏运镜节奏规则

```
前3秒（钩子期）：
  → 必须有1次运镜变化（V-DOWN/V-PUSH）
  → 禁止静止超过2秒

叙事段（3秒-4分30秒）：
  → 运镜每10-15秒至少1次变化
  → 静止镜头单次不超过5秒（竖屏观众耐心更短）
  → 高潮段运镜密度翻倍（每5秒1次变化）

结尾段（最后10秒）：
  → 至少1次V-PULL（拉远消逝感）或完全静止（定格/黑屏）
```

## 三、短剧节奏模板

### 3.1 竖屏短剧三段式（5分钟版）

```
┌──────────────────────────────────────────────┐
│  钩子段 (0:00-0:03)  3秒                      │
│  目标：0.5秒内建立视觉锚点，3秒内制造悬念       │
│  手法：极致单画面/快速运镜/声音先入             │
│  禁忌：慢热开场/建立镜头过长                    │
├──────────────────────────────────────────────┤
│  叙事段 (0:03-4:30)  4分27秒                   │
│  分为2-4个情绪段落，每段30-90秒                  │
│  节奏规则：                                     │
│    · 每段内部：铺垫→升级→小高潮→喘息            │
│    · 段与段之间：5-10秒过渡（黑屏/静默/切光）   │
│    · 情绪曲线总体上行，最后一段是全剧高潮        │
├──────────────────────────────────────────────┤
│  收束段 (4:30-4:55)  25秒                      │
│  目标：制造"必须看下一集"的欲望                 │
│  手法：                                        │
│    · 悬念截断（高潮瞬间切黑）                    │
│    · 反转揭示（最后一帧新信息）                  │
│    · 符号呼应（首集符号在末帧重现）              │
│  禁忌：圆满收尾/缓慢淡出                       │
└──────────────────────────────────────────────┘
```

### 3.2 碎片化时间轴规则

> 竖屏短剧的执行单位不是"场景"而是"碎片"——每个碎片=一个AI可生成片段（10-15秒）。

| 规则 | 说明 |
|------|------|
| **碎片时长** | 10-15秒/碎片（AI单次生成的稳定时长） |
| **碎片粒度** | 每碎片=1个景别+1个运镜+1个画面描述 |
| **时间轴格式** | 入点→出点（如 00:10→00:18） |
| **精度** | 精确到秒，不做帧级标注 |
| **碎片编号** | 段落字母+序号（如 A1/A2/A3 或简写为行号） |
| **段间过渡** | 段与段之间必须有2-5秒过渡碎片（黑屏/切光/静默） |

### 3.3 5分钟短剧碎片分配参考

| 段落类型 | 时长范围 | 碎片数 | 节奏 |
|----------|:--------:|:------:|------|
| 开场/建立 | 20-40秒 | 2-4 | 慢→中（V-DOWN为主） |
| 追逐/行动 | 30-60秒 | 3-5 | 快（手持/跟拍为主） |
| 危机/坠落 | 15-30秒 | 2-3 | 极快→骤停（POV-ROT为主） |
| 沉寂/水下 | 30-60秒 | 3-5 | 极慢（长镜头+微变化） |
| 超自然/灵魂 | 40-80秒 | 4-6 | 慢→中（V-PULL+光效） |
| 仪式/缝合 | 30-50秒 | 3-4 | 庄严慢速（MACRO为主） |
| 重生/高潮 | 20-30秒 | 2-3 | 爆发→定格 |
| 收束/黑屏 | 5-10秒 | 1 | 骤停 |

## 四、竖屏短剧特有符号系统

### 4.1 符号锚点管理

> 竖屏短剧因单集时长短，必须用"符号锚点"建立跨集记忆。

| 符号类型 | 示例 | 管理规则 |
|----------|------|----------|
| **身份物** | 发簪/项链/伤疤 | 每集至少出现1次，首尾必须呼应 |
| **色彩锚点** | 赤红/冥火蓝/金色 | 颜色在分镜表中必须标注色值 |
| **生物符号** | 蝴蝶/鸟/鱼 | 出现→消失→变形=角色命运映射 |
| **自然符号** | 海/悬崖/风 | 环境=情绪外化，不可随意更换 |
| **光影符号** | 瞳孔金点/眼缝透光 | 超自然元素必须跨集一致 |

### 4.2 符号出现节奏

```
EP01: 符号首次出现（建立）
EP02-05: 符号反复出现（强化）
EP06-10: 符号变形/异化（深化）
EP11+: 符号终极呼应（闭环）
```

## 五、竖屏短剧与横屏长片的核心差异速查

| 维度 | 横屏长片(20-120min) | 竖屏短剧(3-5min) |
|------|:-------------------:|:----------------:|
| 开场 | 可慢热3-5分钟 | **0.5秒必须抓住** |
| 场景转换 | 叠化/匹配剪辑 | **硬切/黑屏/声音桥** |
| 台词量 | 大量对话驱动 | **极简/无台词**，画面驱动 |
| 角色数量 | 多角色叙事 | **1-2个核心角色** |
| 情绪曲线 | 三幕式缓慢上行 | **锯齿形急上骤停** |
| 结尾 | 圆满/开放 | **截断式悬念** |
| 音乐 | 连续铺底 | **点状打击**：关键时刻单音/和弦 |
| 静默 | 少用 | **作为情绪武器**：3-5秒完全静默 |
| 观众耐心 | 相对充裕 | **3秒法则**：每3秒需一个注意力锚点 |
