## 使用建议

1. **分镜创作前**：先阅读理论手册，建立系统认知
2. **创作过程中**：参考镜头字典和拆分规则
3. **对标学习**：参考拉片案例，借鉴优秀作品
4. **完成后自检**：使用自检清单确保质量
