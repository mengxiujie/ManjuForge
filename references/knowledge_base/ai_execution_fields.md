# AI执行版分镜字段规范

> 本文档定义"AI执行版"分镜脚本的字段体系，使分镜表直接对接AI视频/图像生成工具，从"人类导演指令"转化为"AI可执行片段"。

---

## 一、设计理念

传统分镜表面向"人类摄制组"——摄影师看景别/运镜，演员看台词/走位，美术看色调/道具。

**AI执行版面向"AI生成器"**——每个碎片必须是独立的、完整的、可闭环的生成指令。

核心原则：
1. **每行=一个AI生成片段**（10-15秒视频或1张关键帧图像）
2. **每行自包含**——AI无需跨行理解上下文即可生成
3. **优先级驱动**——P0先做，P1后补，P2迭代

## 二、AI执行版字段定义

### 2.1 完整字段表

| 字段 | 代号 | 必填 | 说明 | 示例 |
|------|:----:|:----:|------|------|
| 碎片编号 | × | ✅ | 段落+序号 | A3 / B1 / H2 |
| 段落 | 段 | ✅ | 段落标识 | A/B/C/D/E/F/G/H/I |
| 时间轴 | 时间 | ✅ | 入点→出点 | 00:18→00:25 |
| 景别/运镜 | 景 | ✅ | 景别+运镜组合 | WS·垂直下降 |
| 画面描述 | 画 | ✅ | 导演级画面指令（含构图/色调/情绪锚点） | 见下文 |
| 声音设计 | 声 | ✅ | 四层声音分层标注 | 🎵尺八·单音 🔊海浪 💓无 🫧无 |
| AI提示词/优先级 | AI | ✅ | AI生成提示词 + P0/P1/P2 | P0 悬崖场景·晴天竖屏 |

### 2.2 与传统分镜字段对照

| 传统字段 | AI执行版对应 | 变化说明 |
|----------|:----------:|----------|
| 镜号 | 碎片编号 | 从"01/02"改为"段落+序号" |
| 时长 | 时间轴 | 从"5s"改为"00:10→00:18"精确区间 |
| 台词/独白 | 画面描述内嵌 | 竖屏短剧极少台词，如有则嵌入画面描述 |
| 音效/音乐 | 声音设计(4层) | 从简单文本升级为分层标注 |
| 情绪氛围 | 画面描述内嵌 | 情绪=画面描述的一部分，标注为"情绪锚点" |
| 色调建议 | 画面描述内嵌 | 色值直接写入描述（如 深蓝#0a2a5c） |
| —新增— | AI提示词 | 传统无此字段，AI版核心新增 |
| —新增— | 优先级P0/P1/P2 | 传统无此字段，AI版执行优先级 |

## 三、时间轴格式规范

### 3.1 格式

```
入点→出点
MM:SS→MM:SS
```

示例：
- `00:00→00:10` — 首个碎片
- `01:30→01:42` — 中段碎片
- `04:50→04:55` — 末尾碎片

### 3.2 规则

| 规则 | 说明 |
|------|------|
| 碎片间无缝衔接 | 上一个出点=下一个入点（如 00:10→00:18, 00:18→00:25） |
| 过渡碎片可重叠 | 黑屏/切光过渡碎片可与前后碎片重叠1-2秒 |
| 精度到秒 | 不做帧级标注（MM:SS:FF格式禁用） |
| 总时长校验 | 所有碎片时长之和=全集总时长 |
| 黑屏碎片 | 黑屏也占时间轴（如 02:27→02:30） |

## 四、AI提示词字段规范

### 4.1 提示词结构

```
[优先级] 资产/场景名称·变体描述
→ 附件A·资产编号（如有关联资产）
```

示例：
```
P0 悬崖场景·晴天竖屏
P0 女娃人形·近景·正面（绑定三视图）
P1 赤蝴蝶·微距特写·翅膀开合 → 附件A·资产7
P2 无需AI生成·纯黑画面
```

### 4.2 提示词编写规则

| 规则 | 说明 | 正确示例 | 错误示例 |
|------|------|----------|----------|
| 竖屏标注 | 必须注明9:16 | `9:16 vertical format` | 无标注 |
| 分辨率 | 必须注明≥1080P | `1080P minimum` | 无标注 |
| 色值标注 | 关键色彩用HEX | `crimson red #c41e3a` | `red` |
| 风格锚定 | 首行必须锁定风格 | `Chinese ancient mythology style` | 无风格词 |
| 情绪词 | 必须有1个情绪词 | `melancholic` | 无情绪词 |
| 禁止词 | 标注不想要的元素 | `NO text, NO watermark` | 无禁止词 |
| 资产绑定 | 绑定已有资产时标注 | `（绑定三视图）` | 无标注 |

### 4.3 AI提示词英文模板（附录A资产专用）

```
[Style anchor], [Shot type] of [Subject], [Key visual details],
[Color/mood], [Composition], [Technical specs],
9:16 vertical format, [Resolution], [Exclusions]
```

示例：
```
Chinese ancient mythology style, extreme close-up of a young girl's eye,
iris deep navy blue #1a3a6e, pupil rapidly dilating from 30% to 90%,
tears at corner, cinematic macro shot, dark background,
9:16 vertical format, 1080P minimum, NO text, NO watermark
```

## 五、优先级体系（P0/P1/P2）

### 5.1 定义

| 优先级 | 含义 | 时机 | 影响 |
|--------|------|------|------|
| **P0** | 开拍前**必须**就位 | 阻塞制作 | 缺少P0资产=无法开拍 |
| **P1** | 制作前**应**就位 | 影响质量 | 缺少P1资产=可开工但需补 |
| **P2** | 可**后续迭代** | 不阻塞 | 缺少P2资产=上线后补 |

### 5.2 优先级判定规则

| 资产类型 | 默认优先级 | 升级条件 | 降级条件 |
|----------|:----------:|----------|----------|
| 核心角色三视图 | P0 | — | 从不降级 |
| 核心场景 | P0 | — | 从不降级 |
| 关键道具（剧情道具） | P1 | 涉及EP首尾呼应→P0 | 仅背景出现→P2 |
| 生物/特效资产 | P1 | 核心叙事依赖→P0 | 仅1次出现→P2 |
| 背景/环境 | P2 | 主场景→P1 | — |
| 弹幕/互动系统 | P2 | — | 从不升级 |

### 5.3 在分镜表中标注

```html
<span class="p0">P0</span> 女娃人形·近景·正面（绑定三视图）
<span class="p1">P1</span> 赤蝴蝶·微距特写
<span class="p2">P2</span> 弹幕粒子系统
```

### 6.2 资产与碎片绑定规则

```
1. 每个碎片必须绑定至少1个资产（纯黑/切光碎片除外）
2. 每个资产必须标注绑定碎片编号
3. 资产优先级≥碎片依赖的最高优先级
4. 同一碎片绑定多个资产时，标注主资产→辅资产
```

## 七、AI挑战镜头标注

### 7.1 定义

> AI挑战镜头=当前AI视频/图像生成技术难以稳定产出的镜头类型，需特别标注以便：
> 1. 提前预留更多生成迭代次数
> 2. 考虑替代方案或后期合成
> 3. 制作时分配更多资源

### 7.2 挑战等级

| 等级 | 标注 | 说明 | 典型镜头 |
|------|:----:|------|----------|
| 🔴 S级 | `AI-挑战·S` | 当前AI几乎无法稳定产出，需后期合成 | 瞳孔动态四阶段、红线逐针缝合 |
| 🟡 A级 | `AI-挑战·A` | AI可产出但一致性差，需多轮迭代 | 金红重生眼缝透光、沧老蓝眼光芒 |
| 🟢 B级 | `AI-挑战·B` | AI可产出但细节需打磨 | 水下面部光斑、碎光聚集成人形 |

### 7.3 在分镜表中标注

```
P0 瞳孔特写·动态放大（AI-挑战·S）→ 附件A·资产2
P0 金红眼缝透光（AI-挑战·A）→ 附件A·资产10
```

## 八、执行检查清单（AI版新增项）

> 在原有质量自检清单基础上，AI执行版需额外检查以下项目：

| 检查项 | 必填 | 说明 |
|--------|:----:|------|
| [ ] 每个碎片时间轴无缝衔接 | ✅ | 无间隔无重叠（过渡碎片除外） |
| [ ] 每个碎片有AI提示词 | ✅ | 含风格锚定+竖屏标注+分辨率 |
| [ ] 每个碎片有优先级标注 | ✅ | P0/P1/P2 |
| [ ] P0资产全部绑定碎片 | ✅ | 无悬空P0资产 |
| [ ] AI挑战镜头已标注 | ✅ | S/A/B三级 |
| [ ] 色值与符号锚点一致 | ✅ | 关键色彩有HEX值 |
| [ ] 声音设计四层完整 | ✅ | 🎵🔊💓🫧 |
| [ ] 碎片时长在10-15秒范围 | ⚠️ | 超出范围需拆分或合并 |
| [ ] 总时长符合平台限制 | ✅ | 红果≤5分钟 |
| [ ] 结尾无圆满收尾 | ✅ | 竖屏短剧必须悬念截断 |
