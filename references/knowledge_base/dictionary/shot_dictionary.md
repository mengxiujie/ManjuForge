# 镜头字典

本字典定义了动漫分镜中常用的所有镜头类型，包括景别、运镜、特效等，是分镜创作的参考工具。

---

## 第一章：景别字典

### 1.1 基础景别

#### 远景（Wide Shot / Long Shot）
- **定义**：展示广阔空间或场景全貌
- **距离**：人物在画面中非常小
- **时长**：5-15秒
- **用途**：建立环境、展示规模、交代空间关系
- **情绪**：史诗感、孤独、宏大
- **动漫示例**：《进击的巨人》巨人来袭的远景

#### 全景（Full Shot）
- **定义**：人物全身可见
- **距离**：人物充满画面
- **时长**：4-10秒
- **用途**：展示全身动作、舞蹈、战斗
- **情绪**：展示、正式
- **动漫示例**：《鬼灭之刃》剑技展示

#### 中景（Medium Shot）
- **定义**：膝盖以上或半身
- **距离**：人物腰部以上
- **时长**：3-8秒
- **用途**：对话、日常活动
- **情绪**：自然、常态
- **动漫示例**：大多数对话场景

#### 近景（Close-up）
- **定义**：胸部以上
- **距离**：人物占据画面1/2以上
- **时长**：2-5秒
- **用途**：强调表情、内心反应
- **情绪**：亲密、强调
- **动漫示例**：角色情绪变化的特写

#### 特写（Close-up / MCU）
- **定义**：脸部或局部
- **距离**：头部占据画面
- **时长**：1-3秒
- **用途**：极致强调、细节展示
- **情绪**：紧张、强调、冲击
- **动漫示例**：《你的名字》两人对视

#### 大特写（Extreme Close-up / ECU）
- **定义**：眼睛、嘴巴、手等局部
- **距离**：局部占据画面
- **时长**：0.5-2秒
- **用途**：极致细节、关键物品
- **情绪**：极度紧张、关键信息
- **动漫示例**：重要道具、变身瞬间

---

### 1.2 特殊景别

#### 两分法（Two Shot）
- **定义**：两个人物各占画面一半
- **用途**：两人对话、对比
- **情绪**：平衡、对等

#### 过肩镜头（Over the Shoulder）
- **定义**：从一人肩后拍摄另一人
- **用途**：对话、第三方视角
- **情绪**：参与感、对话感

#### 插入镜头（Insert Shot）
- **定义**：突然切入的细节镜头
- **用途**：强调关键物品
- **情绪**：强调、提醒

#### 反应镜头（Reaction Shot）
- **定义**：角色对事件的反应
- **用途**：展示情绪
- **情绪**：共鸣、同理心

## 第二章：运镜字典

### 2.1 固定镜头

#### 固定（Static / Lock Off）
- **定义**：摄像机完全不动
- **用途**：稳定感、强调角色
- **情绪**：平静、压抑、等待

#### 顿住（Hold）
- **定义**：镜头突然静止
- **用途**：强调、停顿
- **情绪**：紧张、震惊

### 2.2 垂直运动

#### 推镜头（Dolly In / Push In）
- **定义**：摄像机向前移动
- **用途**：接近、强调、进入内心
- **情绪**：聚焦、紧张、揭示
- **动漫**：《盗梦空间》推进梦境

#### 拉镜头（Dolly Out / Pull Out）
- **定义**：摄像机向后移动
- **用途**：远离、揭示、结束
- **情绪**：释然、孤独、退场
- **动漫**：《千与千寻》千寻离开汤屋

#### 升降镜头（Crane Up/Down）
- **定义**：摄像机垂直上升/下降
- **用途**：展示高度、全貌
- **情绪**：升华、压迫、释放
- **动漫**：《你的名字》黄昏之时

### 2.3 水平运动

#### 摇镜头（Pan）
- **定义**：摄像机水平/垂直转动
- **用途**：展示空间、跟随、扫描
- **情绪**：观察、自然、揭示
- **动漫**：《哈尔的移动城堡》移动城堡

#### 移镜头（Dolly Pan / Tracking）
- **定义**：摄像机水平移动
- **用途**：展示长距离、跟随
- **情绪**：同行、参与、移动
- **动漫**：《鬼灭之刃》立体机动

#### 跟镜头（Follow Shot）
- **定义**：跟随移动角色
- **用途**：同步感、紧迫感
- **情绪**：紧张、参与、追逐

### 2.4 旋转运动

#### 旋转镜头（Rotate / Spin）
- **定义**：摄像机绕轴旋转
- **用途**：眩晕、天旋地转
- **情绪**：混乱、眩晕、魔法
- **动漫**：《盗梦空间》走廊失重

#### 环绕镜头（Orbit / 360°）
- **定义**：绕角色旋转拍摄
- **用途**：展示关系、戏剧性
- **情绪**：戏剧性、紧张

### 2.5 特殊运镜

#### 甩镜头（Whip Pan / Swish）
- **定义**：快速摇镜头
- **用途**：高速、时间流逝
- **情绪**：速度、冲击、切换

#### 变焦（Dolly Zoom / Vertigo Effect）
- **定义**：推+拉同时进行
- **用途**：创造眩晕、不安
- **情绪**：心理变化、恐惧

#### 第一人称 POV
- **定义**：主观视角
- **用途**：沉浸感、恐怖
- **情绪**：参与、紧张、恐怖

## 第三章：角度字典

### 3.1 水平角度

#### 正面（Front View）
- **定义**：摄像机正对角色
- **用途**：对话、直接表达
- **情绪**：诚实、直接

#### 侧面（Profile）
- **定义**：摄像机从侧面拍摄
- **用途**：展示侧面特征、运动
- **情绪**：客观、叙述

#### 背面（Back View）
- **定义**：摄像机从背面拍摄
- **用途**：跟随、隐藏身份
- **情绪**：神秘、未知

### 3.2 垂直角度

#### 仰视（Worm's Eye / Low Angle）
- **定义**：摄像机从下往上拍
- **用途**：增加压迫感、伟大感
- **情绪**：压迫、伟大、威胁
- **动漫**：《进击的巨人》巨人压迫感

#### 俯视（Bird's Eye / High Angle）
- **定义**：摄像机从上往下拍
- **用途**：展示全貌、增加渺小感
- **情绪**：掌控、渺小、监控
- **动漫**：《鬼灭之刃》战场全景

#### 鸟瞰（Top Shot）
- **定义**：完全垂直向下
- **用途**：地图感、战略感
- **情绪**：战略、全局

### 3.3 倾斜角度

#### 荷兰角（Dutch Angle / Tilting）
- **定义**：摄像机倾斜
- **用途**：不安、紧张、混乱
- **情绪**：紧张、危险、失衡
- **动漫**：《盗梦空间》梦境不稳定感

## 第四章：特殊镜头

### 4.1 动漫特有镜头

#### 速度线
- **定义**：画面边缘的速度线
- **用途**：高速移动、冲击
- **情绪**：速度、力量
- **动漫**：《鬼灭之刃》《火影忍者》

#### 定格帧（Freeze Frame）
- **定义**：画面突然静止
- **用途**：强调、对比
- **情绪**：强调、幽默

#### 效果线
- **定义**：表示冲击、力量的线条
- **用途**：打击感、冲击
- **情绪**：力量、冲击
- **动漫**：《龙珠》龟派气功

#### 内心描写
- **定义**：角色内心可视化
- **用途**：心理表现
- **情绪**：思考、回忆、决策
- **动漫**：《进击的巨人》艾伦内心

#### 回忆闪回
- **定义**：突然回到过去
- **用途**：补充信息、情感触发
- **情绪**：怀旧、揭示
- **动漫**：《你的名字》回忆片段

#### 转场特效
| 名称 | 用途 |
|------|------|
| 淡入淡出 | 时间过渡 |
| 叠化 | 时间流逝 |
| 黑场 | 场景分隔 |
| 帘入帘出 | 空间转换 |

### 4.2 效果镜头

#### 慢动作（Slow Motion）
- **定义**：时间变慢
- **用途**：强调动作、情感
- **情绪**：强调、力量、浪漫
- **动漫**：《鬼灭之刃》爆种瞬间

#### 快动作（Speed Up）
- **定义**：时间加速
- **用途**：表示时间流逝
- **情绪**：幽默、加速

#### 频闪（Stroboscopic）
- **定义**：闪烁效果
- **用途**：梦境、幻觉
- **情绪**：不真实、紧张

## 第五章：镜头组合

### 5.1 常用组合

#### 对话组合
```
[全景建立] → [过肩] → [正反打] → [特写反应]
```

#### 战斗组合
```
[远景建立] → [中景] → [特写] → [动作] → [结果]
```

#### 情感组合
```
[远景环境] → [中景] → [近景] → [特写]
```

### 5.2 情绪镜头序列

#### 建立紧张
```
固定 → 轻微移动 → 快速推 → 特写
```

#### 释放情绪
```
特写 → 拉 → 远景 → 固定
```

#### 突然冲击
```
正常 → 甩镜头 → 定格 → 特写
```

## 第六章：快速查询

### 6.1 情绪对应镜头

| 目标情绪 | 推荐镜头 |
|----------|----------|
| 紧张 | 推镜头、特写、快速切换 |
| 放松 | 拉镜头、远景、固定 |
| 神秘 | 背面、低角度、慢速 |
| 力量 | 仰视、特写、慢动作 |
| 浪漫 | 逆光、特写、慢动作 |
| 恐怖 | POV、俯视、晃动 |
| 搞笑 | 特写、定格、夸张 |
| 悲伤 | 远景、雨景、慢动作 |

### 6.2 类型对应镜头

| 作品类型 | 常见镜头特点 |
|----------|--------------|
| 热血战斗 | 高速切换、特写、速度线 |
| 日常 | 固定、中景、自然 |
| 科幻 | 旋转、特效、远景 |
| 奇幻 | 魔法特效、慢动作 |
| 悬疑 | 暗调、特写、慢速 |
| 爱情 | 特写、逆光、慢动作 |
