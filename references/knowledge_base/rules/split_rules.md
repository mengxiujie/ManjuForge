# 分镜拆分规则

本规则定义了从剧本到分镜的系统化拆分流程，确保分镜的完整性和专业性。

---

## 一、剧本分析阶段

### 1.1 剧本结构拆解

#### 幕结构划分
```
第一幕（建置）：0-25%
  - 引入世界/人物
  - 建立日常
  - 激励事件

第二幕（对抗）：25%-75%
  - 挑战出现
  - 升级冲突
  - 中点转折
  - 最低点

第三幕（解决）：75%-100%
  - 最终挑战
  - 高潮
  - 结局
```

#### 场景清单
- 统计总场景数
- 标注每场戏的类型（室内/室外、日/夜）
- 标记主要角色出场

### 1.2 关键信息提取

| 信息项 | 内容 |
|--------|------|
| 时间线 | 故事发生在什么时间跨度 |
| 地点 | 主要场景清单 |
| 人物 | 出场人物及关系图 |
| 事件 | 主要情节点列表 |
| 情绪曲线 | 全剧情绪起伏 |

## 二、场景拆分规则

### 2.1 场景定义

**场景 = 同一时间 + 同一地点**

```
一个场景结束标志：
1. 时间改变（昼→夜）
2. 地点改变
3. 重要角色离开/新角色进入
4. 场景间隔（黑场、淡入淡出）
```

### 2.2 场景分类

| 场景类型 | 拆分要点 |
|----------|----------|
| 对话场景 | 以镜头切换分隔说话者 |
| 动作场景 | 以动作节点分隔 |
| 情绪场景 | 以情绪转折分隔 |
| 过场场景 | 快速过渡，可合并 |

### 2.3 场景权重分配

根据重要程度分配镜头数量：

| 场景类型 | 镜头密度 | 说明 |
|----------|----------|------|
| 核心场景 | 15-25镜头/分钟 | 高潮、转折点 |
| 重要场景 | 10-15镜头/分钟 | 情节点、人物塑造 |
| 推进场景 | 5-10镜头/分钟 | 剧情推进 |
| 过场场景 | 2-5镜头/分钟 | 时间过渡 |

## 三、镜头拆分规则

### 3.1 镜头拆分原则

#### 基本拆分点
```
1. 景别变化
2. 角度变化
3. 角色动作变化
4. 位置/走位变化
5. 情绪变化
```

#### 拆分信号词
| 信号词 | 拆分提示 |
|--------|----------|
| "突然" | 需要冲击感镜头 |
| "同时" | 需要平行剪辑 |
| "另一方面" | 需要场景切换 |
| "回想起" | 需要闪回 |
| "随着" | 需要节奏变化 |

### 3.2 镜头数量估算

#### 估算公式
```
镜头数 = 场景时长(秒) × 镜头密度(镜头/秒)
```

#### 镜头密度参考
| 场景类型 | 密度 | 示例 |
|----------|------|------|
| 高速战斗 | 3-5/秒 | 鬼灭之刃 |
| 日常对话 | 1-2/秒 | 日常番 |
| 情感高潮 | 1-3/秒 | 你的名字 |
| 悬疑紧张 | 2-4/秒 | 推理番 |

### 3.3 时长分配规则

#### 单镜头时长参考
| 景别 | 时长范围 | 说明 |
|------|----------|------|
| 远景 | 5-15秒 | 建立环境 |
| 全景 | 4-10秒 | 展示人物关系 |
| 中景 | 3-8秒 | 对话/动作 |
| 近景 | 2-5秒 | 表情/反应 |
| 特写 | 1-3秒 | 强调/细节 |
| 大特写 | 0.5-2秒 | 极致强调 |

#### 节奏变化规则
- **安静场景**：长镜头为主
- **紧张场景**：短镜头快切
- **过渡场景**：中等镜头
- **高潮场景**：长短镜头交替

## 四、拆分流程

### 4.1 标准流程

```
Step 1: 通读剧本
  ↓
Step 2: 划分幕/场
  ↓
Step 3: 列出场景清单
  ↓
Step 4: 确定每场情绪
  ↓
Step 5: 分配镜头数量
  ↓
Step 6: 逐镜拆分
  ↓
Step 7: 调整节奏
  ↓
Step 8: 完善细节
```

### 4.2 快速拆分法（紧急情况）

1. **标记关键帧**：找出每场最重要的3-5个画面
2. **填充中间**：用过渡镜头连接关键帧
3. **检查节奏**：确保情绪流畅

## 五、特殊情况处理

### 5.1 对话场景拆分

#### 标准对话分镜
```
镜头1: A说话的中景
镜头2: B反应的近景
镜头3: A说话的反打
镜头4: B说话的近景
...交替
```

#### 特殊对话处理
| 情况 | 处理方式 |
|------|----------|
| 两人边走边说 | 跟随镜头 |
| 打电话 | 交替+特写 |
| 内心独白 | 特写+闪回 |
| 争吵 | 快速交替+特写 |

### 5.2 战斗场景拆分

#### 战斗分镜结构
```
开场：建立双方（远景）
↓
第一轮：试探性攻击（中景）
↓
升级：拿出真正实力（近景+特效）
↓
高潮：终极技能（特效+远景）
↓
结束：结果展示（特写或远景）
```

#### 动作拆分点
```
1. 准备动作（蓄力）
2. 攻击动作（最快）
3. 命中/躲避
4. 后果展示
```

### 5.3 情感场景拆分

#### 情感爆发分镜
```
前兆：细节特写
↓
积累：内心反应
↓
爆发：最大特写+慢动作
↓
延续：结果展示
```

### 5.4 回忆/闪回拆分

#### 闪回结构
```
触发：特写触发物
↓
过渡：淡入/模糊
↓
回忆内容：与主线不同色调
↓
结束：淡出/回到现实
```

## 六、自检清单

### 6.1 拆分自检
- [ ] 每场戏有明确的开始和结束
- [ ] 镜头数量与场景重要性匹配
- [ ] 时长分配合理
- [ ] 景别有变化
- [ ] 运镜有变化
- [ ] 情绪节奏有起伏

### 6.2 常见问题
| 问题 | 解决方案 |
|------|----------|
| 镜头太平淡 | 增加景别变化 |
| 时长太长 | 拆分增加镜头 |
| 节奏单调 | 添加快慢变化 |
| 情绪太平 | 增加特写镜头 |
| 转换生硬 | 添加过渡镜头 |

## 七、参考案例

### 案例：1分钟对话场景拆分

**场景**：两人在咖啡厅讨论重要事情

**镜头分配**（约15-18个镜头）：
```
1. 远景：咖啡厅全景（3s）
2. 中景：两人对坐（3s）
3. 近景：A开口（3s）
4. 特写：B的表情（2s）
5. 中景：B回应（3s）
6. 特写：A的反应（2s）
7. 中景：A拿出文件（3s）
8. 特写：文件特写（2s）
9. 中景：B惊讶（2s）
10. 特写：A的眼神（2s）
11. 远景：窗外经过的人（2s）
12. 近景：两人的手握在一起（3s）
```

**情绪曲线**：
```
平静 → 紧张 → 震惊 → 坚定 → 温暖
```

| 转换类型 | 需要的镜头数 |
|----------|--------------|
