# 风格模板详细指南

> 用户选择风格后，按以下规则调整生成策略。不只是"换个情绪"，而是台词风格、节奏、分镜、色调全面调整。

---

## 1. 甜宠风

### 核心公式
误会 → 解开心结 → 甜蜜互动 → 小危机 → 更甜

### 台词风格调整
- 男主：嘴硬心软，说完狠话偷偷做暖心事
- 女主：表面嫌弃实际心动，会脸红（特写镜头）
- 对话节奏：短句多，反问多，"谁要你……"句式
- 禁止：直接说"我喜欢你"（要靠行动暗示）

### 分镜调整
- 景别：以CU（特写）和MCU（中近景）为主，展示表情变化
- 色调：暖色为主，#FFD700（甜蜜金）#FFB6C1（心动粉）
- 运镜：多用缓慢推进（靠近感）、柔焦（梦幻感）
- 视觉锤：脸红特写、不经意的肢体接触、偷偷看对方的眼神

### 节奏调整
- 前3秒：甜蜜瞬间或误会瞬间（不要暴力/冲突）
- 中段：误会带来的小虐（30-60秒）
- 结尾：甜蜜回归+新的小悬念

### AI提示词风格锚点调整
```
Chinese romance drama, soft warm lighting, pastel color palette,
semi-realistic anime style, dreamy bokeh background,
gentle expressions, blush details, consistent character design
```

## 2. 虐恋风

### 核心公式
误会加深 → 牺牲 → 隐忍 → 爆发 → 遗憾/和解

### 台词风格调整
- 台词少，眼神戏多——"不说"比"说"更虐
- 金句密度高：每集至少1句让观众截图的台词
- 内心独白多于对话
- 禁止：角色主动解释误会（虐恋的核心就是"不说"）

### 分镜调整
- 景别：ECU（大特写）极多——眼泪、攥紧的手、颤抖的嘴唇
- 色调：冷色调为主，#2C3E50（压抑蓝）#8B0000（暗红=痛）
- 运镜：缓慢后拉（孤独感）、V-UP（垂直上升=消逝）
- 视觉锤：眼泪滑落、转身离去的背影、空荡荡的房间

### 节奏调整
- 前3秒：痛苦瞬间（不是暴力，是情感冲击）
- 中段：隐忍堆积
- 结尾：爆发或遗憾截断（不要给圆满结局，留给下一集）

### AI提示词风格锚点调整
```
Chinese melodrama, cold blue-grey tones, dramatic shadows,
semi-realistic anime style, melancholic atmosphere,
tear details, lonely composition, consistent character design
```

## 3. 悬疑风

### 核心公式
伏笔 → 误导 → 线索 → 反转 → 真相（或新悬念）

### 台词风格调整
- 台词信息密度高，每句话都有可能埋线索
- 对话中故意留白（"那天晚上……算了"）
- 角色之间互相试探
- 禁止：旁白直接解释谜底（让观众自己推理）

### 分镜调整
- 景别：大量主观镜头（POV）、窥视感构图
- 色调：暗色调+高对比，#1A1A1A（深黑）#FF4500（危险橙）
- 运镜：手持感（不安定）、快速推近（发现线索）
- 视觉锤：镜子/倒影、门缝中窥视、关键道具特写

### 节奏调整
- 前3秒：一个反常画面（让人想"这是怎么回事？"）
- 中段：线索逐步释放
- 结尾：反转或"差一步就发现真相"的截断

### AI提示词风格锚点调整
```
Chinese mystery thriller, high contrast lighting, dark shadows,
semi-realistic anime style, suspenseful atmosphere,
Dutch angles, shallow depth of field, noir color palette,
consistent character design
```

## 4. 校园风

### 核心公式
日常 → 暗恋 → 误会 → 表白 → 成长

### 台词风格调整
- 口语化，像真实学生说话
- 内心独白多（暗恋的小心思）
- 朋友之间的吐槽和起哄
- 禁止：过于成熟的台词（不符合年龄）

### 分镜调整
- 景别：中景多（展示校园环境+人物互动）
- 色调：明亮清新，#87CEEB（天空蓝）#98FB98（青春绿）
- 运镜：固定为主（日常感）、偶尔跟拍（奔跑/上学路）
- 视觉锤：教室窗户的阳光、操场的背影、传纸条

### 节奏调整
- 前3秒：日常中的心动瞬间（不经意的眼神、手指碰触）
- 中段：青春的小烦恼
- 结尾：甜蜜或遗憾的小截断

### AI提示词风格锚点调整
```
Chinese school campus drama, bright natural lighting, soft pastel colors,
semi-realistic anime style, youthful innocent atmosphere,
school uniform details, cherry blossom or sunlight elements,
consistent character design
```

## 5. 古风

### 核心公式
权谋/情感 → 抉择 → 牺牲 → 家国/爱情抉择

### 台词风格调整
- 半文半白，不要太文言（观众听不懂）
- 尊称/谦称要准确（"殿下""在下""奴家"）
- 金句用对仗/排比（"我以江山为聘，你以天下为棋"）
- 禁止：现代网络用语（"绝绝子""yyds"）

### 分镜调整
- 景别：大量WS（远景）展示场景气势+ECU（大特写）展示情绪
- 色调：暖金+深红，#8B6914（暖金）#C0392B（正红）
- 运镜：缓慢后拉（孤独感）、升降（权力感）
- 视觉锤：灯笼/烛光侧脸、令牌/玉佩道具、长廊背影

### 节奏调整
- 前3秒：强视觉冲击（不是暴力，是仪式感/冲突瞬间）
- 中段：权谋或情感的张力
- 结尾：悬念或金句截断

### AI提示词风格锚点调整
```
Chinese historical drama, warm golden lighting, rich color palette,
semi-realistic anime style, traditional Chinese architecture,
detailed hanfu fabric textures, candlelight atmosphere,
consistent character design
```

## 6. 玄幻风

### 核心公式
修炼 → 战斗 → 突破 → 逆袭 → 更强的对手

### 台词风格调整
- 短句多，战斗中的怒吼/招式名
- "废物逆袭"的台词弧线：前期低声忍耐→后期霸气宣言
- 反派的嘲讽要够狠（让观众想看打脸）
- 禁止：过于复杂的战术解说（观众看不懂）

### 分镜调整
- 景别：WS（远景展示战斗规模）+ECU（大特写展示力量爆发）
- 色调：热血红+能量蓝，#FF4500（火焰橙）#4169E1（能量蓝）
- 运镜：快速剪辑（战斗感）、V-DOWN（垂直下降=压制感）
- 视觉锤：能量爆发瞬间、瞳孔变化、地面碎裂

### 节奏调整
- 前3秒：战斗/力量展示的高光瞬间
- 中段：劣势→蓄力→反击
- 结尾：突破或更强敌人出现的截断

### AI提示词风格锚点调整
```
Chinese fantasy action, dynamic dramatic lighting, vibrant energy colors,
semi-realistic anime style, power aura effects, particle effects,
intense battle composition, speed lines, impact frames,
consistent character design
```

## 使用方法

用户选择风格后：
1. 用对应风格的AI提示词锚点替换默认锚点
2. 调整台词风格（按"台词风格调整"规则）
3. 调整分镜景别和色调（按"分镜调整"规则）
4. 调整节奏（按"节奏调整"规则）
5. 选择合适的视觉锤（从"视觉锤"列表中选2-3个）
