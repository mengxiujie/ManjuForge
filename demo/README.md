## 记忆机制演示

最简单的验证方式：打开 `第1集_完整版.md` 和 `第6集_完整版.md`，对比以下元素的跨集变化：

- **沈舟敲桌子**：桌面→膝盖→屏风边缘→门框+腰间+桌面→椅子背
- **"bug"口头禅**：私下宣言→当众宣言→自嘲
- **林霜空间距离**：门外→门口→面前→并肩→身边
- **林霜家主令牌**：目光追→攥紧→站起可见→主动不戴
- **赵天赐策略**：暗讽→提亲→扬言→经济封锁→精准断供→供应链封锁
