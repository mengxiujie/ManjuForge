## 📖 项目介绍

AI Drama Storyboard Skill 是一款面向短视频漫剧工业化量产的专业级AI Agent技能工具。

**核心能力**：一句话需求/小说片段/长文本 → 完整商用漫剧剧本 + 专业分镜脚本

区别于市面仅能生成"纯文字文案"的普通AI工具，本项目实现：

- **剧本生成**：自动拆解剧情、生成人设、结构化剧本、爆点预埋
- **分镜输出**：支持横屏16:9标准版 + 竖屏9:16 AI执行版双模式
- **专业深度**：内置13个世界级拉片案例、竖屏运镜字典、声音四层设计
- **全链路适配**：输出内容可直接对接AI绘图、AI配音、自动剪辑全流程

适配抖音、小红书、快手、视频号全平台短剧节奏，专为短视频完播率、互动率优化。

- **竖屏专项规范**：构图规则/运镜字典/节奏模板/符号系统
- **AI执行字段**：时间轴/提示词/P0-P2优先级/资产清单/挑战镜头

### 📈 短视频流量优化

- 自动预埋悬念、反转、情绪冲突
- 3秒钩子设计（前3秒必须建立视觉锚点）
- 碎片化时间轴（10-15秒/碎片，总时长≤5分钟）
- 悬念截断式结尾（提升追更率）

### 🎨 六大风格模板

1. **甜宠风**：温柔治愈、互动甜蜜、节奏轻快
2. **虐恋风**：情绪压抑、误会冲突、隐忍遗憾
3. **悬疑风**：伏笔密集、反转铺垫、层层递进
4. **校园风**：青春清新、日常真实、互动青涩
5. **古风风**：古韵雅致、场景唯美、台词古韵
6. **玄幻风**：特效感强、场景宏大、设定奇幻

### 🔗 全平台兼容部署

- **Hermes Agent**：原生支持 delegate_task 任务分发，可多技能并行流水线作业
- **OpenClaw / ClawHub**：完全符合平台上架规范，支持一键部署、市场收录
- **硅基流动 SiliconFlow**：完美兼容 GLM-4.7-Flash、GLM-4-9B 等免费模型
- **Cursor / 本地私有化部署**：兼容OpenAI通用接口，自定义模型自由适配

## 🚀 核心功能

### 1. 智能剧本生成
- 一句话需求 → 完整剧本结构
- 小说片段 → 漫剧改编剧本
- 长文本 → 多集连续剧剧本

### 2. 自动人设生成
- 批量生成角色外貌、性格、关系、台词风格
- 统一人物设定，避免人设崩塌

### 3. 结构化分镜输出
- 横屏标准版：镜号/场景/景别/运镜/时长/画面描述/台词/情绪/色调
- 竖屏AI执行版：碎片编号/段落/时间轴/景别运镜/画面描述/声音设计/AI提示词

### 4. AI提示词自动生成
- 每幕配套专业构图、光影、风格提示词
- 可直接用于AI绘图出图

### 5. 声音四层设计
- 🎵 音乐层：情绪定调+主题动机
- 🔊 环境层：空间感知+氛围营造
- 💓 生理层：角色内在状态（心跳/呼吸）
- 🫧 细节层：微观质感+触觉感

### 6. AI挑战镜头标注
- 🔴 S级：AI几乎无法稳定产出，需后期合成
- 🟡 A级：AI可产出但一致性差，需多轮迭代
- 🟢 B级：AI可产出但细节需打磨

### 7. 批量剧集迭代
- 支持多集批量生成
- 自动承接前剧人设、剧情、风格
- 解决断更、人设崩塌问题

## ⚙️ 快速安装使用

### 1. 仓库克隆

```bash
git clone https://github.com/xxx/ai-drama-storyboard-skill.git
```

### 2. 平台部署

将项目文件夹导入对应AI Agent平台，加载 `skill.json` 配置文件即可启用。

### 3. 快速使用

**输入**：任意剧情素材/创作需求，选择对应漫剧风格

- 完整结构化漫剧剧本（含人设、场景、台词、音效）
- 专业分镜脚本（横屏标准版 或 竖屏AI执行版）
- AI绘图提示词（可直接用于出图）
- 资产清单（P0/P1/P2优先级）
